The following big alignment was produced by first running the following FASTA command 

  fasta -q -d 30 mgmstm1.aa s > fasta_out.30
where "mgmstm1.aa" is one of the example sequences from the FASTA distribution and "s" is the FASTLIBS entry for the Swiss-Prot database (Version 30). Then, the following fmtseq command was executed: 
  fmtseq fasta_out.30 -fpretty -nametop -nameleft=11 -width=60 -idprefix=sp
         -nocolspace -interline=3 '-gapout= ' -otemp
where the actual command was give on a single line containing all of the options.
